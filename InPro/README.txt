NorthShield Contents - Static Website Package

What you get:
- Fully static HTML/CSS/JS site (no frameworks)
- Responsive design
- SVG images generated in-house (no third-party licensing needed)
- Pages: Home, Solutions, Process, Industries, About, Contact, Submit a File

How to run locally:
1) Open index.html in a browser.

How to publish:
- Upload the entire folder to any web host (Netlify, Cloudflare Pages, cPanel, S3, etc.)
- Or import pages into a builder (you may copy/paste sections).

To connect forms:
- Contact and Submit a File currently open mailto links.
- Your developer can connect to a form backend (Netlify Forms, Formspree, custom endpoint) and add file upload.

Branding:
- Replace "NorthShield Contents" and placeholder emails/phone numbers in pages/contact.html and pages/submit-claim.html.
